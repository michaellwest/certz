# AI Prompts

Prompts for AI assistants working on certz development and maintenance.

## Available Prompts

| Prompt | Purpose |
|--------|---------|
| [new-command.md](new-command.md) | Template for implementing new CLI commands |
| [new-test.md](new-test.md) | Template for creating test suites |
| [release.md](release.md) | Checklist for preparing releases |
| [future-work.md](future-work.md) | Summary of completed modernization work |

## Usage

These prompts provide context for AI assistants to understand the project's patterns, constraints, and completed work. They complement the main `CLAUDE.md` file in the project root.

### When to Use Each Prompt

- **new-command.md** - When adding a new CLI command (creates command, service, options, result, tests)
- **new-test.md** - When writing tests for existing or new functionality
- **release.md** - When preparing a new version for release
